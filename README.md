# YouTube Desktop Downloader

A desktop application for searching YouTube and downloading videos locally with a clean and modern UI.  
Built with PySide6 and powered by `yt-dlp`.

This project focuses on speed, minimalism, and usability — no playback, no clutter, just downloading.

-----------------------------------------------------------------------------

## Features

- Search YouTube directly inside the app  
- Paste a YouTube link to download instantly  
- Choose any folder as the download location  
- Video quality options:
  - Fast (360p)
  - Balanced (720p)
  - Max (best available)
- Real-time download progress tracking  
- Clean dark UI inspired by modern apps  
- No ads, no tracking, no background services  

-----------------------------------------------------------------------------

## Tech Stack

- Python  
- PySide6 (Qt for Python)  
- yt-dlp  

-----------------------------------------------------------------------------

## Requirements

- Python 3.9+  
- PySide6  
- yt-dlp  

-----------------------------------------------------------------------------

## Installation

Install dependencies:
- pip install PySide6 yt-dlp    (in CMD administrator)
- Now download the actual file inside of the "releases" tab
- Finally open it and have fun
